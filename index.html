Web MIDI API

  • The Web Audio API and the Web MIDI API
  • integrate analogue synthesis and MIDI to the web
  • These APIs are mere footnotes compared to our everyday work, but they're very powerful
  • So to explore these technologies, let's make a song
  • Let's make Daft Punk's 1997 single, Around the World
  • For those in the audience, here's what it sounds like
  • Where do we start? Let's start with this, my Arturia KeyStep Pro
  • This is a MIDI controller and a sequencer. It looks like a keyboard, but it's not
  • This makes no sound of its own, it sends MIDI data to my computer
  • MIDI data tells computers, synthesisers, drum machines what notes to play
  • MIDI data is even used to control lighting and visual effects as well
  • MIDI is message passing, and nothing else

Arturia KeyStep Pro

  • Let's hook this up to our browser using the Web MIDI API
  • Then we can see what MIDI data looks like.
  • MIDI sends a lot of data, but the essential core is a set of three values
  • The first value is the status byte, often used to identify the control
  • Some controls have different values depending on context, such as keyup or keydown
  • The second is the data byte, which carries information such as the note pressed
  • Additional parameters follow, such as the velocity or the range of a knob
  • You don't need API documentation to use these values, and weirdly it makes them easier to understand
  • They are designed to be learned intuitively, and professional audio software works the same way
  • Now that we know how to tell the computer to make a noise, we need to figure out how to make a noise
  • This is what the Web Audio API is for.
  • The workflow for the web audio API works by connecting nodes together

Patching a modular synthesiser

  • Oscillators create an electronic signal that make noise when you connect them to a loudspeaker
  • I've hooked my synthesiser up to an oscilloscope
  • Plugin to show you the shape of the sound they make
  • Sine waves sound soft
  • Sawtooth waves sound sharp
  • Square waves sound like a mix of both
  • Pulse waves can vary their pulse width to sound sharper or squarer
  • Changing the shape changes the texture of the sound

What the Web does best

  • So what does the web do best?
  • Should we be thinking of integrating the web audio API into our sites?
  • Is it a replacement for "serious" production tools?
  • Well, no. Enterprise packages like Ableton and Logic
  • will give you higher quality tools and technology out of the box
  • That doesn't mean web audio and web MIDI are worthless
  • There is a huge community of hobbyist and experimental musicians, including circuit benders.
  • Circuit benders make electronic music using scrap, toys, and broken synthesisers.
  • If they can make music out of rubbish, you can bet someone can make something cool with a web browser
